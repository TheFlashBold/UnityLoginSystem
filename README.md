Requirements:
- MongoDB (https://www.mongodb.com/download-center#community)
- nodejs (https://nodejs.org/en/download/) LTS is fine
	
Steps:
- run "npm i" in the backend folder
- create the structure "C:\data\db", where data and db are folders
